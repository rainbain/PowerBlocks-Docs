\hypertarget{known_issues_autotoc_md75}{}\doxysubsection{\texorpdfstring{Power\+PC 750CL SIMD Support}{Power\+PC 750CL SIMD Support}}\label{known_issues_autotoc_md75}
The “paired-\/singles” SIMD instructions on the Power\+PC 750CL are an IBM extension to the standard 750 core. LLVM’s Power\+PC backend does not currently implement support for these instructions.

They are documented on\+: \href{https://wiibrew.org/wiki/Paired_single}{\texttt{ https\+://wiibrew.\+org/wiki/\+Paired\+\_\+single}}

Past attempts to upstream support were made years ago, but the LLVM backend has changed significantly since then, so those patches are no longer compatible.

Long-\/term goals\+:
\begin{DoxyItemize}
\item Add paired-\/single support to LLVM
\item Investigate whether auto-\/vectorization for paired-\/singles is feasible
\end{DoxyItemize}
