\hypertarget{windows_autotoc_md65}{}\doxysubsection{\texorpdfstring{Prerequisites}{Prerequisites}}\label{windows_autotoc_md65}
Power\+Block\textquotesingle{}s runs inside of a "{}\+Linux like"{} environment. In order to provide this MSYS2 must be installed from \href{https://www.msys2.org/}{\texttt{ here}}.

From the MSYS2 shell, install the required build tools. 
\begin{DoxyCode}{0}
\DoxyCodeLine{pacman\ -\/S\ cmake\ ninja\ wget\ unzip}

\end{DoxyCode}


\DoxyHorRuler{0}
 \hypertarget{windows_autotoc_md67}{}\doxysubsection{\texorpdfstring{Download the SDK}{Download the SDK}}\label{windows_autotoc_md67}
From the MSYS2 shell, download and unzip the latest version of the SDK for Windows from the \href{https://github.com/rainbain/PowerBlocks/releases}{\texttt{ Power\+Blocks Releases Page}}.


\begin{DoxyCode}{0}
\DoxyCodeLine{cd\ \string~}
\DoxyCodeLine{wget\ https://github.com/rainbain/PowerBlocks/releases/download/<VERSION>/PowerBlocks-\/windows.zip}
\DoxyCodeLine{unzip\ PowerBlocks-\/windows.zip\ }
\DoxyCodeLine{rm\ PowerBlocks-\/windows.zip}

\end{DoxyCode}
 \DoxyHorRuler{0}
 \hypertarget{windows_autotoc_md69}{}\doxysubsection{\texorpdfstring{Start a Project}{Start a Project}}\label{windows_autotoc_md69}
We will create a project folder and copy in the {\ttfamily Hello\+World} example.


\begin{DoxyCode}{0}
\DoxyCodeLine{mkdir\ MyProject}
\DoxyCodeLine{cd\ MyProject}
\DoxyCodeLine{cp\ -\/r\ \string~/PowerBlocks-\/windows/examples/HelloWorld/.\ .}

\end{DoxyCode}
\hypertarget{windows_autotoc_md70}{}\doxysubsubsection{\texorpdfstring{Build the Project}{Build the Project}}\label{windows_autotoc_md70}
To build it we will run Power\+Blocks\textquotesingle{} export script to add the compilers and toolchain to the path and build the project.


\begin{DoxyCode}{0}
\DoxyCodeLine{mkdir\ build}
\DoxyCodeLine{cd\ build}
\DoxyCodeLine{.\ \string~/PowerBlocks-\/windows/export.sh}
\DoxyCodeLine{cmake\ ..}
\DoxyCodeLine{ninja}

\end{DoxyCode}


You should now see a {\ttfamily Hello\+World.\+elf} inside the {\ttfamily build/} folder of your project.
