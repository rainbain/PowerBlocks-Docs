\doxysection{Linux Installation}
\hypertarget{linux}{}\label{linux}\index{Linux Installation@{Linux Installation}}
Power\+Blocks provides downloadable releases for Linux.

\DoxyHorRuler{0}
 \begin{DoxyNote}{Note}
These instructions may be incomplete. Please report any missing steps or issues to the \href{https://github.com/rainbain/PowerBlocks/issues}{\texttt{ Issue Tracker}}
\end{DoxyNote}
\hypertarget{linux_autotoc_md58}{}\doxysubsection{\texorpdfstring{Prerequisites}{Prerequisites}}\label{linux_autotoc_md58}
To begin, install the required build tools using your package manager of choice.


\begin{DoxyCode}{0}
\DoxyCodeLine{sudo\ apt\ install\ cmake\ ninja-\/build\ wget\ unzip}

\end{DoxyCode}


\DoxyHorRuler{0}
 \hypertarget{linux_autotoc_md60}{}\doxysubsection{\texorpdfstring{Download the SDK}{Download the SDK}}\label{linux_autotoc_md60}
Then download and unzip the latest version of the SDK for Linux from the \href{https://github.com/rainbain/PowerBlocks/releases}{\texttt{ Power\+Blocks Releases Page}}.

Replace {\ttfamily \texorpdfstring{$<$}{<}VERSION\texorpdfstring{$>$}{>}} with the latest version. 
\begin{DoxyCode}{0}
\DoxyCodeLine{cd\ \string~}
\DoxyCodeLine{wget\ https://github.com/rainbain/PowerBlocks/releases/download/<VERSION>/PowerBlocks-\/linux.zip}
\DoxyCodeLine{unzip\ PowerBlocks-\/linux.zip\ }
\DoxyCodeLine{rm\ PowerBlocks-\/linux.zip}

\end{DoxyCode}


\DoxyHorRuler{0}
 \hypertarget{linux_autotoc_md62}{}\doxysubsection{\texorpdfstring{Start a Project}{Start a Project}}\label{linux_autotoc_md62}
We will create a project folder and copy in the {\ttfamily Hello\+World} example.


\begin{DoxyCode}{0}
\DoxyCodeLine{mkdir\ MyProject}
\DoxyCodeLine{cd\ MyProject}
\DoxyCodeLine{cp\ -\/r\ \string~/PowerBlocks-\/linux/examples/HelloWorld/.\ .}

\end{DoxyCode}
\hypertarget{linux_autotoc_md63}{}\doxysubsubsection{\texorpdfstring{Build the Project}{Build the Project}}\label{linux_autotoc_md63}
To build it we will run Power\+Blocks\textquotesingle{} export script to add the compilers and toolchain to the path and build the project.


\begin{DoxyCode}{0}
\DoxyCodeLine{mkdir\ build}
\DoxyCodeLine{cd\ build}
\DoxyCodeLine{.\ \string~/PowerBlocks-\/linux/export.sh}
\DoxyCodeLine{cmake\ ..}
\DoxyCodeLine{ninja}

\end{DoxyCode}


You should now see a {\ttfamily Hello\+World.\+elf} inside the {\ttfamily build/} folder of your project.
