Notes\+:
\begin{DoxyItemize}
\item Each of the inputs, A-\/D, are selectable.
\item Inputs are truncated to 8 bits, except for D that is a signed 11 bits, \mbox{[}-\/1024, 1023\mbox{]}
\item The addition is input D, additional bias, then scale is set to be 0.\+5, 1, 2, or 4
\item If clamping is enabled, color values are capped \mbox{[}0, 255\mbox{]}, otherwise \mbox{[}-\/1024, 1023\mbox{]}
\item You can save the output to 1 of 4 TEV registers, or just use the tev previous register.
\item TEV previous becomes the pixels color at the end of the TEV pipeline.
\item TEVs default mode is biasing mode, it can be in compare mode too. No mixing, It just says, if operation on input A and B is true, pass C, otherwise, 0. It still adds D as a bias.
\end{DoxyItemize}
