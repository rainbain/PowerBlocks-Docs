\chapter{Getting Started}
\hypertarget{getting_started_index}{}\label{getting_started_index}\index{Getting Started@{Getting Started}}
This document is intended to help you setup a Power\+Blocks SDK installation on your machine. After that a short "{}\+Hello World"{} example will be shown to help build an executable for the first time and run it on the Wii.\hypertarget{getting_started_index_autotoc_md39}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{getting_started_index_autotoc_md39}
Power\+Blocks is a SDK providing the compilers, runtime, and various hardware libraries needed for Wii development.

\DoxyHorRuler{0}
\hypertarget{getting_started_index_autotoc_md41}{}\doxysection{\texorpdfstring{What You Will Need}{What You Will Need}}\label{getting_started_index_autotoc_md41}
It is recommended to have an installation of \href{https://dolphin-emu.org/}{\texttt{ Dolphin Emulator}} for easy testing.

You will want a Wii with the ability to launch your own applications. For setting up a modded Wii, read the \href{https://wii.hacks.guide/}{\texttt{ Wii Hacks Guide}}.

An SD card will be needed as it is recommended to launch Power\+Blocks generated executables from the Wii\textquotesingle{}s SD card slot for filesystem support.

 \hypertarget{getting_started_index_autotoc_md48}{}\doxysection{\texorpdfstring{Launching Executables}{Launching Executables}}\label{getting_started_index_autotoc_md48}
The example project created from the installation option of choice generates {\ttfamily Hello\+World.\+elf}.

This can be launched on both emulator and Wii hardware for testing.

\begin{DoxyNote}{Note}
Currently, the Homebrew Channel is required to launch applications on emulator. Power\+Blocks currently does not fully initialize the hardware and needs a launcher in front of it to do this.
\end{DoxyNote}
